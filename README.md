# 仿网易严选项目——前端

# 🌀 功能
- 首页 : 广告banner,推荐分类,推荐商品
- 分类功能 : 2级分类,分类浏览商品
- 搜索功能 : 关键字搜索,搜索历史,热门关键字,
- 浏览商品详情 :  商品信息,2级评论,卖家信息,相关商品,标记为收藏,标记为想要
- 其他功能 : 用户主页,查看自己发布,收藏,卖出,买到的商品
- 订单,支付 : **未完成**
